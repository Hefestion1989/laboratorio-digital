# Academic Semantic Search

Buscador academico local para indexar y consultar documentos con busqueda semantica.

## Proposito

Este proyecto esta pensado para trabajar con materiales academicos locales, especialmente carpetas sincronizadas desde Drive o documentos descargados para estudio.

## Estado

Proyecto local funcional, conservado dentro de `laboratorio-digital` como herramienta de investigacion.

## Componentes

- App de busqueda local.
- Scripts de instalacion y ejecucion en Windows.
- Indexado de documentos y busqueda por conceptos.

## Nota publica

La version publica evita subir entornos virtuales, indices, logs y datos locales. Esos archivos deben generarse en cada computadora.
