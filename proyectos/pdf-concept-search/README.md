# PDF Concept Search

Prototipo simple para buscar conceptos dentro de PDFs.

## Proposito

Explorar busqueda local sobre documentos PDF y organizar fragmentos utiles para lectura o investigacion.

## Estado

Prototipo conservado dentro de `laboratorio-digital`.

## Nota publica

No se suben PDFs ni datos locales. Cada usuario debe agregar sus propios documentos en su copia local.
