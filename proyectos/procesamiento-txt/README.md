# Procesamiento de archivos TXT

Flujo simple para recibir archivos `.txt`, procesarlos y dejar resultados auditables.

## Estructura esperada

- `entrada/`: archivos originales.
- `salida/`: resultados generados.
- `logs/`: registros de ejecucion.
- `scripts/`: automatizaciones de PowerShell.

## Nota publica

La version publica evita subir entradas, salidas y logs reales. Esas carpetas se generan o completan localmente.
