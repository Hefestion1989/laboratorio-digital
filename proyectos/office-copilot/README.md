# Office Copilot

Kit de macros para usar ChatGPT desde Excel y Word.

## Incluye

- `ExcelCopilot.bas`: macros para analizar celdas, generar formulas o sugerir automatizaciones.
- `WordCopilot.bas`: macros para reescribir, resumir o generar texto desde una seleccion.
- `Set-OpenAIKey.ps1`: helper para guardar la API key como variable de entorno local.

## Nota de seguridad

No subir claves privadas ni archivos con credenciales. La API key debe configurarse en cada computadora.
