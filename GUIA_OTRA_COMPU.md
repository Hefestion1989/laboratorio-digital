# Guia para usar este laboratorio en otra computadora

## Opcion simple

1. Entrar a GitHub.
2. Abrir el repositorio `laboratorio-digital`.
3. Descargarlo o clonarlo.
4. Abrir esa carpeta en la app donde vayas a trabajar.

## Para que sea tu base de trabajo

Cada vez que empieces un proyecto nuevo, crealo dentro de:

```text
laboratorio-digital/proyectos/
```

Si es solo una prueba, usalo dentro de:

```text
laboratorio-digital/experimentos/
```

## Sobre el inicio por defecto

El repositorio no puede obligar a una app instalada en otra computadora a abrirlo automaticamente. Lo mas cercano es abrir manualmente `laboratorio-digital` la primera vez; muchas apps recuerdan la ultima carpeta usada.
